0. intro

frf-pane - это программа, которая предназначена для комфортного офлайнового просмотра записей с фрифидика
Программа работает совместно с программой backfrf и ставится в тот же каталог, где размещена программа backfrf

1. настройка

В файле настроек pane.ini можно задать следующие параметры работы программы:
debug, port

Описание этих параметров - ниже в пункте 5

2. запуск

2.1. индексирование

frf-pane build feedname
или
frf-pane build @myname
(имя фида берется из настроек)

Для подключения FTS необходимо использовать ключ i:
frf-pane -i build feedname

2.2. запуск сервера

frf-pane server feedname
или
frf-pane server @myname
(имя фида берется из настроек)

3. просмотр

После запуска сервера просмотр выполняется в браузере по адресу http://localhost:3000/ (по умолчанию).
Если номер порта изменялся в настройках, то просмотр будет по адресу http://localhost:NNNN/, где NNNN - номер заданного порта

4. просмотр доступных фидов

frf-pane list all

5. параметры настроек

Пользовательские параметры хранятся в файле pane.ini в секции [default]:
debug - режим записи логов о ходе работы программы
Возможные значения:
0 - не записывать логи
1 - записывать логи

port - номер порта, на котором будет работать встроенный веб-сервер
Возможные значения:
:NNNN - вместо NNNN цифровой номер порта, сервер доступен по всем сетевым интерфейсам
hostname:NNNN - вместо NNNN цифровой номер порта, сервер доступен по hostname
